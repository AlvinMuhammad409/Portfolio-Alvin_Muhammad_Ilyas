Hypertext Markup Language is a standard markup language for documents designed to be displayed in internet browsers. This can be helped by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript and VBScript.

CSS

CSS is a language used for describing the presentation of a document written in HTML or XML (including XML dialects like SVG or XHTML). In simpler terms, CSS defines how HTML elements are displayed on a web page, including their layout, colors, fonts, sizes, and more.
